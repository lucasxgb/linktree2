# Linktree 2.0 (Curso React Js com TypeScript do zero ao avançado na pratica)

Projeto desenvolvido no Curso de React Js com TypeScript do zero ao avançado na pratica, com o objetivo de criar um fluxo semelhante ao do linktree, o mesmo foi desenvolvido utilizando a biblioteca React junto com o TailwindCss. Como banco de dados para garantir a persistência dos dados utilizou-se o Firebase Store. Esse projeto oferece as seguintes funcionalidades:
- Login;
- Cadastro de Redes Sociais;
- Cadastro de Links;
- Remoção de Links;
- Visualização de todos os links na página home.
